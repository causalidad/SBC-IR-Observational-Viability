# SBC/F3 Phase IV Final Report

## Executive Summary

The SBC/F3 program has reached the stage of an observationally constrained cosmological phenomenology.

The objective of this phase was to determine whether the SBC/F3 infrared sector could remain compatible with the principal cosmological observations while introducing controlled perturbative modifications.

The analysis incorporated:

- Planck-inspired TT, EE and TE spectral diagnostics
- Matter power spectrum consistency tests
- fσ8 growth observables
- DESI DR2 BAO constraints
- Pantheon+ full covariance supernova distances
- Global posterior reconstruction

---

# Main Observational Result

A stable IR-safe region emerges repeatedly across independent analyses.

Preferred region:

epsilon ≈ 0.060

k0 ≈ 0.015 – 0.022

n ≈ 1.0

The persistence of this region across multiple observables suggests that it is not a numerical artifact but an emergent phenomenological feature of the SBC/F3 framework.

# Background Constraints

A phenomenological background deformation was introduced:

F(z) = -ε exp[-(k0(1+z))^n]

with an effective background coupling α_bg.

Results:

Pantheon+ alone:
- allows mild positive α_bg

BAO:
- strongly constrains α_bg

Joint BAO + Pantheon+:
- selects α_bg ≈ 0

Therefore:

The viable SBC/F3 branch preserves the ΛCDM background expansion history.
